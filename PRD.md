# PRD: 아고라 블로그 기반 실행형 자기계발 서비스

## 개요(Introduction)
- 아고라는 Paul Graham 에세이를 한국 20-30대 PM·초기 창업자 맥락으로 번역·재구성해 블로그(롱폼)로 발행하고, 카드뉴스/릴스/노트 숏폼을 함께 제공하는 실행형 자기계발/브랜딩 서비스다.
- USP: “실행 번역” — 깊이 있는 사고 도구를 한국 스타트업 실무 언어로 재해석하고, 바로 실행 가능한 템플릿·미니 과제·체크리스트·리마인더·CTA를 일관된 블로그 포맷에 담아 제공.

## 해결하려는 문제(Problem Statement)
- 사용자는 PG 사고 도구를 실무에 적용하고 싶지만 영어 원문이 추상적이고 한국 스타트업 문법으로의 실행 가이드가 없어 시간을 낭비하며 실행·브랜딩 성과를 얻지 못한다.

## 목표 및 성공 지표(Goals & Success Metrics)
- 목표: PG 사고 도구의 “실행 번역”을 통해 시간 절약과 즉시 실행 편의를 제공하고 실행/브랜딩 성과를 높인다.
- 정량 지표
  - 콘텐츠 저장/공유율: +20%p vs 베이스라인
  - 팔로워→DM 상담 리드 전환율: ≥3%
  - 실행 체크리스트 1개 이상 완료율(7일 내): ≥30%
  - 주간 재방문율(WAU/MAU): ≥45%
  - AI 추천 노출 대비 실행 클릭률: ≥10%
  - 블로그 CTA 클릭률(체크리스트 다운로드/DM): ≥8%, 블로그→IG/뉴스레터 역유입 비율 트래킹
- 정성 지표
  - “실행 가이드 명확성” 만족도 +20 (NPS 스타일)
  - “시간 절약 체감” 응답 ≥70%

## 사용자 페르소나(User Personas)
- Persona A: 28세 PM(경력 3년, 시리즈A). 필요: 전략 사고 확장, 팀 공유용 요약, 빠른 실행 체크리스트, 모바일 친화 숏폼.
- Persona B: 32세 초기 창업자(팀 5명). 필요: 주간 실행 미션, 우선순위 설정, 리마인더, 팀 영감 스토리, CTA 통해 상담/템플릿 확보.
- Persona C: 26세 주니어 기획자. 필요: 숏폼 학습, 아이데이션 트리거, 미니 과제 템플릿, SEO로 발견되는 블로그 롱폼.

## 기능 요구사항(Feature Requirements)
- 블로그 포맷 및 발행 파이프라인
  - 표준 블로그 구조 고정: 요약(맥락/핵심 문장) → 한국 스타트업 실무 번역 → 실행 체크리스트(5분·20분·1시간) → 사례/툴킷 → CTA(체크리스트 다운로드/DM 상담).
  - 워크플로우: PG 롱폼 정리 → 카드뉴스/릴스용 스니펫 자동 추출 → 블로그 본문에 임베드/링크 → 게시 전 톤/스타일 가이드 검수(“실행 번역” 유지).
  - 발행 스케줄: 주 3회 정기 + 시사/신규 PG 글 스팟 발행, 월간 테마(집중/탐구/집요함 등)로 시리즈화.
  - SEO/유입 최적화: 제목에 “PG 에세이 한국어 해석 + 핵심 키워드”, 메타디스크립션에 “실행 템플릿” 강조; OG 태그/구조화 데이터 기본 제공.
  - UTM/소스 태깅: IG/뉴스레터/블로그 간 링크 소스 자동 태깅, CTA 버튼별 트래킹.
- 실행 체크리스트/미니 과제
  - 직무·스테이지 기반 체크리스트 자동 생성(PM/그로스/초기 MVP 등)과 난이도/시간 태그(5분·20분·1시간).
  - 완료 체크, 진행도 저장, 캘린더/푸시 리마인더 연동.
  - CTA로 체크리스트 다운로드(파일/노션 링크) 및 DM 상담 유도.
- 콘텐츠 멀티모달 변환/배포
  - PG 에세이→카드뉴스/릴스/노트 멀티모달 요약 생성, 블로그 본문에 스니펫 임베드.
  - 톤/스타일 가이드 적용(한국 스타트업 실무 언어) 및 휴먼 에디트 워크플로우.
  - 주 3회 정기 발행 스케줄러와 스팟 발행 지원.
- 개인화 추천
  - 목표·직무·경력 입력 기반 콘텐츠/체크리스트 추천.
  - 추천 노출→클릭→완료 퍼널 트래킹, A/B 실험 슬롯 제공.
- 커뮤니티 Q&A 보조
  - 중복 질문 감지·요약, AI 초안 답변 후 멘토 보정 UI.
  - 베스트 Q&A를 노트화해 블로그 라이브러리에 편입.
- 트래킹/지표
  - 저장/공유/클릭/완료/DM 유입/CTA 클릭/스크롤 깊이 이벤트 로깅, 소스 태깅.
  - 대시보드: 저장·공유율, 실행 완료율, DM 리드 전환율, 재방문율, CTA 클릭률, 채널별 역유입 비율.
- 운영 도구
  - 프롬프트 템플릿 관리(톤/금지어/PG 핵심 원리).
  - 콘텐츠 승인 플로우(초안→리뷰→게시), 발행 캘린더 관리.

## 출시 범위에서 제외되는 것(Out of Scope)
- 유료 과금/결제 플로우(베타는 무료).
- 팀 단위 협업(다인 편집, 권한, SSO).
- 풀스택 커뮤니티 기능(게시판, 실시간 채팅).
- 고도화된 게이미피케이션(배지/레벨링).
- 오프라인 이벤트/코호트 운영 기능.
